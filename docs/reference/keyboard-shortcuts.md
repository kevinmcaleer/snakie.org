# Keyboard shortcuts

`Cmd` is the modifier on macOS; use `Ctrl` on Windows and Linux. Menu items show
their own shortcut next to them. This page lists the shortcuts confirmed in
Snakie; many standard editor shortcuts (from the OS **Edit** and **View** menus)
also apply.

## Application

| Action | Shortcut |
| --- | --- |
| Open / focus the **Board View** | `Cmd/Ctrl` + `Shift` + `B` |
| Toggle full screen | `Ctrl` + `Cmd` + `F` (macOS) / `F11` (Win/Linux) |
| Zoom UI in / out / reset | `Cmd/Ctrl` + `+` / `-` / `0` |
| Reload window | `Cmd/Ctrl` + `R` |
| Toggle Developer Tools | `Cmd/Ctrl` + `Alt` + `I` |
| Close window | `Cmd/Ctrl` + `W` (windows that support it) |

## Editor

| Action | Shortcut |
| --- | --- |
| **Find** | `Cmd/Ctrl` + `F` |
| **Replace** | `Cmd/Ctrl` + `H` |
| Close the current tab | `Cmd/Ctrl` + `W` |
| Undo / Redo (text) | `Cmd/Ctrl` + `Z` / `Cmd/Ctrl` + `Shift` + `Z` |
| Cut / Copy / Paste / Select all | `Cmd/Ctrl` + `X` / `C` / `V` / `A` |

```{note}
**Run** and **Stop** are toolbar buttons (see [Run and stop code](../how-to/run-and-stop)). The
[Find & Replace](../how-to/find-replace) window opens with `Cmd/Ctrl` + `F` /
`Cmd/Ctrl` + `H` and closes with `Esc`.
```

## Part Editor

When the [Part Editor](../tutorials/author-a-part) is open:

| Action | Shortcut |
| --- | --- |
| **Undo** | `Cmd/Ctrl` + `Z` |
| **Redo** | `Cmd/Ctrl` + `Shift` + `Z` or `Cmd/Ctrl` + `Y` |
| Delete the selected object | `Delete` or `Backspace` |
| Multi-select objects | `Shift`- or `Cmd/Ctrl`-click (or drag a marquee) |
| Resize freely / disable snapping while dragging | hold `Cmd/Ctrl` |
| Close the editor (one level at a time) | `Esc` |

## Board View

| Action | Shortcut |
| --- | --- |
| Close the window | the native title-bar control, or `Esc` |
| Zoom | the `−` / `+` controls, or the mouse wheel |
| Pan | drag the empty canvas |
